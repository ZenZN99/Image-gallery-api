Image Gallery API is a back-end API developed using Bun, TypeScript, Express.js, and MongoDB. It allows users to create accounts, log in, upload images to Cloudinary, edit, delete, like, and view private or public images.

The project is designed to offer a system similar to Pinterest for sharing and interacting with images, with a focus on high performance and ease of maintenance.
